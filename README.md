# ACNH Discord Bot
Discord bot for Animal Crossing: New Horizon

## Tom Nook Bot

### Commands

Item lookup:
```
!item giant teddy bear
```

Villager lookup:
```
!villager raymond
```

### Usage

To install dependencies:
```
pip3 install -r requirements.txt
```

To run:
```
python3 tom_nook_bot.py
```
